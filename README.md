# cybooks

## Installation

1. Télécharger le driver (https://we.tl/t-1owldLt0kY)
2. Dans IntelliJ aller dans File>Project Structure
3. Aller dans Modules
4. Aller dans Dependencies
5. Ajouter (avec le +) le fichier .jar
6. Faire Apply puis Ok
